นโยบายความเป็นส่วนตัว (Privacy Policy)

อัปเดตล่าสุด: 7 มกราคม 2026

1. ข้อมูลที่เราเก็บรวบรวม

แอพพลิเคชัน N8N Order Notification ให้ความสำคัญกับความเป็นส่วนตัวของคุณ เราเก็บรวบรวมข้อมูลเฉพาะที่จำเป็นต่อการทำงานของระบบแจ้งเตือนเท่านั้น ได้แก่:

  • ข้อมูลคำสั่งซื้อพื้นฐาน (เช่น หมายเลขคำสั่งซื้อ, ยอดรวม, สถานะสินค้า) เพื่อส่งการแจ้งเตือน
  • Token การเข้าถึง (Access Token) เพื่อเชื่อมต่อ API (ข้อมูลนี้จะถูกเก็บรักษาอย่างปลอดภัย)

2. วิธีที่เราใช้ข้อมูลของคุณ

เราใช้ข้อมูลที่รวบรวมเพื่อวัตถุประสงค์ในการ "ประมวลผลและส่งการแจ้งเตือน" เท่านั้น เราไม่มีนโยบายนำข้อมูลของคุณไปวิเคราะห์เพื่อการโฆษณา หรือขายข้อมูลให้กับบุคคลที่สาม

3. การเปิดเผยข้อมูล

เราจะไม่เปิดเผยข้อมูลส่วนบุคคลของคุณต่อบุคคลภายนอก เว้นแต่จะได้รับความยินยอมจากคุณ หรือเป็นการปฏิบัติตามข้อกำหนดทางกฎหมาย

4. ความปลอดภัยของข้อมูล

เราใช้มาตรการรักษาความปลอดภัยที่เหมาะสมเพื่อป้องกันการเข้าถึง การเปลี่ยนแปลง หรือการเปิดเผยข้อมูลโดยไม่ได้รับอนุญาต

5. ติดต่อเรา

หากคุณมีคำถามเกี่ยวกับนโยบายความเป็นส่วนตัวนี้ สามารถติดต่อผู้พัฒนาได้ที่อีเมลของผู้พัฒนาแอพพลิเคชันนี้
